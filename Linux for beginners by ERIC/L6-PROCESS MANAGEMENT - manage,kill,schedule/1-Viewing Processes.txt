===== Viewing Processes ====
how to view the prcesses running currently

run ps command

linuxofpratham@HP-LapyMachine:~$ ps
    PID TTY          TIME CMD
    339 pts/0    00:00:00 bash
    471 pts/0    00:00:00 ps

LINUX KERNEL the inner core of operating system that controls everything it assign unique process ID called PID to each process sequentially as process created.

Need to remember PID no which is more impt than just process name.

== ps aux ===
This command tells more details then ps only. 
we’d like to know about processes run by other users and by the system in the background. Without this information, we won’t understand much of what is actually taking place on our system. 

Running the command with the
ps option aux
will display all processes running on the system for all users.

==== impt columns to know in ps process ====

USER - The user who invoked/started the process
PID - The process ID
%CPU - The percent of CPU this process is using
%MEM - The percent of memory this process is using
COMMAND - The name of the command that started the process


====== Filtering by Process Name =====
we usually don’t want every single process displayed on the screen. we want to find information about a single process. how to do that

=== ps aux | grep <what u want to find> ===

below i want to find any process related to python running
linuxofpratham@HP-LapyMachine:~$ ps aux | grep python

=== Find process which using most resources using top command ===
If u run ps command, it display process started in sequences but what if we want to check currently running processes using high memory or cpu usuage.

it displays the processes ordered by resources used, starting with the largest.
Unlike the ps command, which gives us a one-time snapshot, toprefreshes
the list every three seconds by default. This lets you monitor those resource hungry
processes.

System administrators often keep top running in a terminal to 1 dedicated monitor the
use of process resources.
especially if you have multiple tasks running on your system. While you
have running, pressing the H or ? key will bring up a list of interactive
top commands, and pressing Q will quit top.

when top is running, press H for diff option menu , press q to quit
